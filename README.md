## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run mock api on http://127.0.0.1:8125/
```
npm run mock-api
```

### Lints and fixes files
```
npm run lint
```
